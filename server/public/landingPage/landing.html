Desenvolvimento
sem fricção

IoTMaker é uma plataforma de desenvolvimento feita por um maker para a comunidade maker. Você liga componentes e a ferramenta gera o código para você.

O portal é feito para permitir o compartilhamento de ideias e código entre a comunidade.

Pegue um código complexo, gere um bloco e compartilhe com a comunidade.

Atenção: ainda não estamos prontos, ainda estamos em desenvolvimento e validando algumas ideias junto a comunidade.

Esta é a primeira versão e ela gera apenas código golang porque eu sou um desenvolvedor golang.
