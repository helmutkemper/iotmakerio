# Add — Addition

Adds two integer values and sends the result to the output.

## Ports

| Port   | Direction | Type | Description               |
|--------|-----------|------|---------------------------|
| inputX | Input     | int  | First operand             |
| inputY | Input     | int  | Second operand            |
| output | Output    | int  | Result of inputX + inputY |

## Tip

The output can be connected to multiple devices simultaneously.
