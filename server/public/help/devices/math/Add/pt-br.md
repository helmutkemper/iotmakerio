# Add — Adição

Soma dois valores inteiros e envia o resultado para a saída.

## Portas

| Porta  | Direção | Tipo | Descrição                    |
|--------|---------|------|------------------------------|
| inputX | Entrada | int  | Primeiro operando            |
| inputY | Entrada | int  | Segundo operando             |
| output | Saída   | int  | Resultado de inputX + inputY |

## Dica

A saída pode ser conectada a múltiplos dispositivos ao mesmo tempo.
